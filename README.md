# 前言

欢迎来到基于SSM的宿舍报修管理系统的开源项目。本项目致力于为高校宿舍管理人员及学生提供一个便捷、高效的报修平台。通过使用Java、Spring、Spring MVC、MyBatis等主流技术，以及Vue等前端技术，实现了宿舍报修管理的完整流程。

# 内容介绍

本项目主要包含以下功能模块：用户注册、登录、报修申请、维修进度查询、维修评价等。系统管理员可以实时查看报修申请，分配维修任务，跟踪维修进度，并对完成的维修任务进行验收。学生用户则可以方便地提交报修申请，查询维修进度，并对维修服务进行评价。
